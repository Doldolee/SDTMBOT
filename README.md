- Turtlebot driving through reinforcement learning

### 알고리즘
- MobilNet
- SSD
- Q-learning
- feedback

### 환경
- coral
- IOT
- ubuntu

### 작동
0. 강화학습을 통한 주행을 이용해 장애물을 회피하며 이동
1. 광각카메라를 통한 실시간 얼굴인식
2. 얼굴인식 완료시 화상카메라 가동
3. 광각카메라의 좌표데이터 송신 및 전처리 후 온도 영역 추출
4. 화상카메라 온도 체크
5. 일정 온도 이상시 경고 출력
